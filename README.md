# Dotfiles 

## Install new device

### 1: Install repo
Clone repo to hidden directory & run submodules

```
git clone git@github.com:bartvosman/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
git submodule init
git submodule update
```

### 2: Run install bash
Run Install.sh

```
chmod +x ~/.dotfiles/install.sh && ~/.dotfiles/install.sh
```
